WORDLE

For one of my projects in college, I had to re-create the famous game of Wordle in Python using the Guizero Python library.

Here are some screenshots of the game being played:


The rules of the game are that the user has to guess what five letter word has been randomly generated. The user then has to guess what this word is by e.g. entering common words.

If the letter is correct but in the wrong place, the letter tile will go yellow. If the letter is correct and in the right place, it goes green. If the letter is not in the word at all, the letter tile goes red.

STUDY SPACER

For a coursework project in college, I had to design an application for students to help them with planning their study plans.

Here are screenshots of the app I made using Thunkable:

I used colours such as light blue and the image of a panda to give the app a calm and relaxing effect (which is most definitely needed for students).

The app has a timetable feature, in which you can enter what subjects you will study and topics within these subjects. The app will then generate a timetable. The user can also colour code how confident they felt on a topic. There is also a note taking function in which the user can create notes for their subjects.
